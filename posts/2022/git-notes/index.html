commit 规范

标识 描述
feat 新功能feature
fix 修补bug
docs 开发文档变动
style 代码格式变动，无代码变动，比如格式化了代码
refactor 重构业务，既不是新增功能，也不是修复bug
test 增加测试用例
chore 脚手架工具变动，无业务代码变动
