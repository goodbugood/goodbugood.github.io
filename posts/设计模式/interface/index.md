```
Java的设计模式接口没有固定的包，它们可以在不同的包中找到。常见的设计模式接口包括：

java.util.Observer 和 java.util.Observable：观察者模式
java.util.Iterator：迭代器模式
java.util.Map.Entry：迭代器模式中的迭代元素
java.lang.Cloneable：原型模式
java.io.Serializable：原型模式和备忘录模式
java.util.concurrent.Callable：命令模式
java.util.concurrent.Future：命令模式和代理模式
java.util.concurrent.Executor：命令模式和责任链模式
javax.servlet.Filter：责任链模式
除了以上常见的接口外，还有许多其他的接口可以用于实现不同的设计模式。
```
