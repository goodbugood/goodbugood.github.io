# Sublime


### 常用替换正则

#### 中文排版指北

- 多个空格替换成一个空格

- 在中文和英文之间增加空格

```sh
// 搜索
([\x{4e00}-\x{9fa5}]{1})([0-9a-zA-Z]{1})
// 替换
$1 $2
```

```sh
// 搜索
([0-9a-zA-Z]{1})([\x{4e00}-\x{9fa5}]{1})
// 替换
$1 $2
```

- 去除中文之间的空格

```sh
// 搜索
([\x{4e00}-\x{9fa5}]{1}) ([\x{4e00}-\x{9fa5}]{1})
// 替换
$1$2
```
